该 APP 是用 Kotlin 写的 Android APP，主要用来练手。网络数据接口来源[易源接口](https://www.showapi.com)。

***

第一个页面主要功能是展示今日的宜忌趋向以及各个时刻的吉凶方位

第二个页面主要展示今日的星座和运势，以及根据选项查看其它时间的运势

第三个页面是关于周公解梦，根据关键字显示符合的主题，点击查看详情
